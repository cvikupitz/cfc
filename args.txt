Usage: ./cfc 'REGEX' [OPTIONS]
Recursively scans and outputs all files that match REGEX.

  -A, --check-all             Do not ignore files/directories starting with '.'
  -D, --include-directories   Includes directories in the search
  -F, --file-properties       Displays the matching file/directories properties
  -I, --inverse-match         Outputs all results that do NOT match the specified regex
  --max-depth=N               Do not search more than N subdirectories for each
                                directory specified in the search path
  --search=DIR                Adds the directory DIR to the search path
  --threads=N                 Perform the scan with N number of PThreads

  -?, --help                  Displays this help message then exits
  --version                   Dispalys the currently installed version
